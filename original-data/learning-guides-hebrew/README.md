# מדריכי למידה מקיפים ל-BMAD-METHOD

> **המדריך המרכזי** לכל עובד בארגון ללמידה ושליטה מלאה במתודולוגיית BMAD

---

## 📌 ברוכים הבאים!

מערכת המדריכים הזו נבנתה במיוחד כדי לאפשר לכל עובד בארגון - בכל תפקיד, רמת ניסיון ורקע טכנולוגי - ללמוד ולשלוט במתודולוגיית BMAD-METHOD.

**מה תמצאו כאן:**

- 📚 **11 מדריכים כלליים** - למידה מהירה, מילון מונחים, מדריכי אג'נטים ווורקפלואים ועוד
- 👥 **10 מדריכים לפי תפקידים** - מותאמים אישית לכל תפקיד בארגון
- 🎓 **תכניות הדרכה מובנות** - יום 1, שבוע 1, חודש 1
- 🔧 **מדריכי ניהול ותחזוקה** - עדכונים, שדרוגים וניהול תצורה
- 🎨 **עזרים ויזואליים** - דיאגרמות, תרשימי זרימה ומפות מושגים
- ⚡ **כרטיסי עזר מהירים** - מידע חיוני בהישג יד

## 🚀 מסלול למידה מהיר (15 דקות)

**רוצים להתחיל מהר?** עקבו אחרי השלבים הבאים:

1. **קראו**: [00-התחלה-מהירה-לכולם.md](./00-התחלה-מהירה-לכולם.md) - התחלה מהירה ל-15 דקות
2. **התקינו**: פשוט הריצו `npx bmad-method@alpha install` 
3. **נסו**: טענו את אחד האג'נטים והריצו את הפקודה הראשונה
4. **המשיכו**: עברו למדריך התפקיד שלכם למידע מעמיק יותר

## 💡 טיפים להצלחה

### 💪 תרגול עושה את המאסטר

- **אל תקראו בלבד** - נסו כל דוגמה בעצמכם
- **התחילו קטן** - משימה פשוטה ראשונה
- **תרגלו באופן קבוע** - 15 דקות ביום עדיפות על 2 שעות פעם בשבוע
- **אל תפחדו לטעות** - זה חלק מהלמידה

### 🤝 למדו ביחד

- **מצאו חבר ללמידה** - למדו בזוג או בקבוצה
- **שתפו ידע** - תעדו את הלמידה שלכם
- **שאלו שאלות** - אין שאלות טיפשיות
- **עזרו לאחרים** - ללמד אחרים מחזק את הידע

### 📝 תעדו את המסע

- **רשמו הערות** - מה עבד, מה לא
- **צרו דוגמאות משלכם** - מהפרויקטים שלכם
- **בנו מאגר ידע אישי** - רעיונות, טריקים, פתרונות
- **שתפו עם הצוות** - עזרו לכולם להשתפר
