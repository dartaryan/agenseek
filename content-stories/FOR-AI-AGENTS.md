## 📋 תהליך עבודה - 7 צעדים

### צעד 1: קבל הוראה מהמשתמש
המשתמש יגיד לך משהו כמו:
- "צור את המדריך core/glossary"
- "תכין את onboarding/day1"
- "אני רוצה את roles/product-managers"

### צעד 2: קרא את המסמכים הבסיסיים
```bash
קרא את:
1. MASTER-PROMPT.md - הפרומפט המלא שלך (חובה!)
2. blocks-reference.md - כל 14 הבלוקים
3. story-template.md - המבנה הסטנדרטי
```

### צעד 3: מצא ו קרא את הסטורי
```bash
קרא את: stories/{category}/{slug}.md
דוגמה: stories/core/glossary.md
```

**הסטורי מכיל:**
- מטרת המדריך
- קהל יעד
- מבנה מומלץ מפורט
- בלוקים רלוונטיים
- הנחיות כתיבה
- Checklist

### צעד 4: קרא דוגמה קיימת
```bash
קרא לפחות אחד מאלה:
- src/content/locale/he/guides/core/quick-start.json
- src/content/locale/he/guides/roles/developers.json
- src/content/locale/he/guides/agents/intro-pm-analyst.json
```

**למה?** כדי לראות:
- פורמט JSON מדויק
- סגנון כתיבה
- שימוש בבלוקים
- אורך ומורכבות

### צעד 5: צור את המדריך
```bash
צור קובץ JSON חדש:
src/content/locale/he/guides/{category}/{slug}.json
```

**חייב לכלול:**
1. metadata (id, slug, title, description, category, difficulty, estimatedMinutes, icon, tags)
2. tableOfContents (נבנה מה-headings)
3. content (array של ContentBlocks)

### צעד 6: עדכן את הקטלוג
```bash
עדכן את: src/content/locale/he/guides/index.json
```

**הוסף את המטא-דאטה של המדריך החדש למערך.**

### צעד 7: בדוק ואמת
```bash
בדיקות:
1. ה-JSON תקין (אין syntax errors)
2. metadata.id זהה ל-slug
3. metadata.slug זהה לשם הקובץ
4. tableOfContents תואם להדינגים
5. כל heading יש לו id ייחודי
6. כל הבלוקים תקינים
7. הטקסט בעברית
```

## 📝 סגנון כתיבה

### שפה:
- **עברית שוטפת** - כל הטקסט
- **משפטים קצרים** - 10-20 מילים

- **קוד עובד** - לא סתם דוגמאות
- **FAQ** - 5-8 שאלות
- **קישורים** - למדריכים אחרים

## 🚨 שגיאות נפוצות

### 1. JSON לא תקין
```
❌ שכחתי פסיק
❌ סוגריים לא סגורים
❌ מרכאות לא סגורות
✅ בדוק עם JSON validator
```

### 2. בלוקים לא תקינים
```
❌ type לא קיים
❌ field חובה חסר
❌ ערך לא תקין
✅ עקוב אחר blocks-reference בדיוק
```

### 3. ToC לא תואם
```
❌ anchor לא תואם id של heading
❌ level לא נכון
❌ שכחתי heading
✅ בנה אוטומטית מההדינגים
```

### 4. מטא-דאטה לא תואמת
```
❌ id ≠ slug
❌ slug ≠ filename
❌ category לא תקינה
✅ בדוק הכל פעמיים
```
