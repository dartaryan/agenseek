## 📐 מבנה מומלץ

### 1. פתיחה
```
- Heading (H1): כותרת המדריך
- Callout (info): תקציר של 2-3 שורות
- Divider
```

### 2. מבוא (H2: "מבוא")
```
- Text: מה נלמד?
- Text: למי זה מיועד?
- Text: מה צריך לדעת מראש?
- List (unordered): דרישות קדם (אם יש)
```

### 3. סקשן 1 (H2)
```
- Heading (H2): {נושא 1}
- Text: הסבר כללי
- Code או List: דוגמאות
- Callout: נקודות חשובות
```

### 4. סקשן 2 (H2)
```
- Heading (H2): {נושא 2}
- Text: המשך הסבר
- [בחר בלוק מתאים]
```

### 5. סקשן 3 (H2)
```
- Heading (H2): {נושא 3}
- [תוכן רלוונטי]
```

### 6. דוגמאות מעשיות (H2: "דוגמאות מעשיות")
```
- Heading (H3): דוגמה 1
- Text + Code: דוגמה מלאה
- Heading (H3): דוגמה 2
- Text + Code: דוגמה מתקדמת
```

### 7. שאלות נפוצות (H2: "שאלות נפוצות")
```
- Accordion: רשימת שאלות ותשובות (5-8 שאלות)
```

### 8. סיכום ומה הלאה (H2)
```
- Text: סיכום מה למדנו
- Callout (success): נקודות מפתח
- Text: מה הלאה?
- List: מדריכים קשורים (קישורים)
```

## 🎨 בלוקים רלוונטיים

**חובה להשתמש:**
- ✅ Heading (H1, H2, H3)
- ✅ Text
- ✅ Callout (info)

**מומלץ להשתמש:**
- ✅ List
- ✅ Code
- ✅ Accordion (FAQ)
- ✅ Divider

**לשקול להשתמש:**
- Tabs (אם יש תוכן לפי תפקיד)
- Table (אם יש נתונים מובנים)
- Grid + Card (אם יש השוואה)

**לא רלוונטי:**
- Chart (אלא אם יש סטטיסטיקות)
- Image/Video (אלא אם יש חומר ויזואלי)

## 📝 הנחיות כתיבה

### סגנון:
- **עברית שוטפת** - משפטים קצרים וברורים
- **גוף שני** - "אתם תלמדו..." (לא "המשתמש ילמד...")
- **פעיל** - "צרו..." (לא "ייווצר...")
- **פשוט** - הימנע מז'רגון מיותר

### מבנה:
- **H1** - רק אחד (כותרת ראשית)
- **H2** - 5-8 סקשנים עיקריים
- **H3** - תתי-סקשנים לפי הצורך
- **פסקאות** - 3-5 משפטים

### תוכן:
- **דוגמאות** - כל מושג מורכב צריך דוגמה
- **קוד** - קוד מלא ועובד (לא סתם דוגמה)
- **Callouts** - למידע חשוב, אזהרות, טיפים
- **FAQ** - 5-8 שאלות נפוצות

---

## 🔑 נקודות מפתח

### מה חייב להיות במדריך:
1. **מבוא ברור** - מה נלמד, למי זה מיועד
2. **דוגמאות קוד** - לפחות 2-3 דוגמאות מעשיות
3. **נקודות חשובות** - callouts למידע קריטי
4. **FAQ** - שאלות נפוצות עם תשובות
5. **סיכום** - מה למדנו ומה הלאה

### מה לא לשכוח:
- ✅ Table of Contents (נבנה אוטומטית מההדינגים)
- ✅ id ייחודי לכל heading
- ✅ טאגים רלוונטיים (3-6 טאגים)
- ✅ זמן קריאה נכון (estimatedMinutes)
