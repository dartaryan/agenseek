### 2. המקרים שנכללו

#### מקרה 1: Gaming Platform
- **תיאור:** פלטפורמת משחקים מרובי משתמשים Real-time
- **אתגר:** Performance, latency, state management
- **טכנולוגיה:** Node.js, WebSockets, Redis, Kubernetes
- **לקחים:** Architecture-first thinking, client-side prediction

#### מקרה 2: Microservices Migration
- **תיאור:** מעבר ממונוליט Java ל-microservices
- **אתגר:** המערכת קריטית, zero downtime
- **טכנולוגיה:** Node.js, NestJS, RabbitMQ, PostgreSQL
- **לקחים:** Strangler Fig Pattern, incremental migration

#### מקרה 3: Legacy Modernization
- **תיאור:** שדרוג מערכת COBOL בת 35 שנה
- **אתגר:** אף אחד לא מבין את הקוד, סיכון עצום
- **טכנולוגיה:** COBOL → Node.js, API Gateway, CDC
- **לקחים:** Reverse engineering, Shadow Mode, Golden Master Testing

#### מקרה 4: Multi-Team Project
- **תיאור:** 5 צוותים עובדים על אותו מוצר
- **אתגר:** תיאום, communication, dependencies
- **טכנולוגיה:** React, NestJS, React Native, Kubernetes
- **לקחים:** Structured communication, dependency tracking

#### מקרה 5: Global Distributed Team
- **תיאור:** צוות על פני 4 יבשות, 5 time zones
- **אתגר:** Async work, no common hours
- **טכנולוגיה:** Documentation-first, Async workflows
- **לקחים:** Follow-the-sun, RFC process, written standups
