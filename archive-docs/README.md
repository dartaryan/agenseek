## Quick Reference

### Finding Story Documentation
1. **Story specs:** `v1-stories/specs/story-X.X.md`
2. **Implementation summaries:** `v1-stories/specs/*-implementation-summary.md`
3. **Test plans:** `v1-stories/testing/`
4. **Completed stories:** `v1-stories/completed/`

### Finding Planning Documents
1. **Epic docs:** `v1-planning/epics/`
2. **Sprint plans:** `v1-planning/sprints/`
3. **Story catalog:** See `docs/story-catalog.md` (kept in active docs)

### V1 Technical Reference
For architecture and design system documentation, see the active `docs/` folder:
- `docs/architecture.md`
- `docs/ux-design-specification.md`
- `docs/design-system-guide.md`
